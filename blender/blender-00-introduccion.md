# Introducción
Esto es lo basico del modo edición: 

- `Shift + Click` de rueda de raton. Para mover la posicion del la camara.

- Taclado numerico para cambiar el modo de vista

- Teclas `1`, `2` y `3`: Para cambiar el modo de seleccion: bordes, vertices o caras

- Teclas `X`, `Y`, `Z`: Para establecer dimencion en la que se movera/redimensionara/extruira/..etc lo seleccionado.

- Tecla `A`, para seleccionar todo.

- `G`: Mover

- `S`: Escalar

- `E`: Extruir

- `R`: Rotar/Girar

- `F`: Crear un cara, cara o borde, a partir de lo seleccionado
