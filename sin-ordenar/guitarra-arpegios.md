# Guitarra arpegios

Los arpegios es otra forma de tocar la guitarra, es buena para dominar los dedos locos.

Se vera como arpegiar con los dedos.

Con la pua, solo puedo mencionar, que simplemene arrastra la pua por las cuerdas, de tal forma que suenen independientes y no como acorde.

Dedos:
```
p = pulgar
i = indice
m = medio
a = anular
```

# Cómo se usa
El pulgar (p) se encarga de las tres cuerdas graves (6, 5 y 4).
El índice, medio y anular se encargan de las tres agudas (3, 2 y 1).

En un arpegio típico, tocas algo como:
`p–i–m–a–m–i` o `p–i–m–a` (según el patrón o el ritmo).

# Forma de arpegiar todas las cuerdas
```
p-p-p-i-m-a

6-5-4-3-2-1
```


## Ejemplos, patrones de arpegio
Todas las cuerdas
```
p-p-p-i-m-a
___________
6-5-4-3-2-1
```

Completo en abanico (subida y bajada total)
```
p-p-p-i-m-a-m-i-p
_________________
6-5-4-3-2-1-3-4-5
```

Folk / country
```
p-i-p-m
_______
6-3-4-2
```

Clasico lineal
```
p-i-m-a-m-i
___________
6-3-2-1-2-3
```

Folk, pop
```
p-i-m-a-p-i-m-a
_______________
6-3-2-1-5-3-2-1
```
