## `boot`
Incluye todo lo necesario para que funcione el proceso del arranque. Esta carpeta es usada mientras se inicia el sistema operativo. Aca se almacenan los gestores de aranque, como **GRUB**.

**GRUB** guarda todo lo chido aca. Configuración, librerias, programillas, todo eso.

Este directorio sirve para el uso del programa de gestión de inicio del sistema operativo.
