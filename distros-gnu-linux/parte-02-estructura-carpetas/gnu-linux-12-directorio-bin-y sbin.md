## `bin`
Aca se almacenan todas las funciones basicas a nivel de usuario. Solo almacena ejecutables de usuario.

Estos son algunos de los ejecutables que almacena: `cat, cd, cp, grep, gzip, kill, ls, mv, rm, ping, su, ps, tar, vi` y etc.




## `sbin`
Todos los binarios necesarios para tareas administrativas gestionadas por el usuario root o super usuario del sistema se encuentran en el directorio sbin.

Este tambien almacena ejecutables, pero ejecutables que solo se pueden usar con permisos root. Sirven para gestionar tareas del sistema operativo. Funciones internas.
