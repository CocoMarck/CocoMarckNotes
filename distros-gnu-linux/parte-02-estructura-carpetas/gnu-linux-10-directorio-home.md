# Directorio HOME

Es el directorio de los usuarios estándar, y por lo tanto, el destinado a almacenar todos los archivos del usuario, como documentos, foto, vídeos, música, plantillas, etc.

Dentro `/home` hay los directorios personales de todos los usuarios, nombrados según el nombre de usuario utilizado. Así por ejemplo, si en un sistema pongamos que hay dos usuarios denominados User1 y User2, la estructura será así:
- `/home/User1`
- `/home/User2`

Cada directorio de usuario contiene asimismo diferentes carpetas para ayudarlo a clasificar la información. Estas generalmente son: `/Documentos`, `/Imágenes`, `/Música`, `/Plantillas` y `/Vídeos`.

Así se veria el arbol completo:
```
/home/user/
    /Documentos
    /Descargas
    /Imágenes
    /Música
    /Plantillas
    /Vídeos
    /Vídeos
```



Eso es todo.
