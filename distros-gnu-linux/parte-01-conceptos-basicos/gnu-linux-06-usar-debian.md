# Empezar a usar una distro base Debian.
Ahora si ya mucho quilombo. Vamos a empezar a usar una distro GNU-Linux base Debian.

Usaremos una distribución basada en Debian. Su diseño visual se parece a windows.

- Q4OS: https://www.q4os.org/index.html

Nombre complicadete pero ta chido. Este OS viene con varios sabores/versiones, una mas pesada que la otra, pera no complicarnos la vida yo recomiendo la mas ligerona:
Trinity Live x64: <https://www.q4os.org/dnt02.html>

Tienes que asegurarte de que tu pc es x64 bits, y si no jala el link pos busca la trinity Live x64, en la parte de descargas, en la pagina web.

**Tendremos que ver un video de como instalar Q4OS.**

**O podemos ir a su documentación de como instalar**: <https://www.q4os.org/dqa018.html>

O las dos cosas al mismo tiempo.


Una vez instalado pos vemos como anda la cosa, como se ve, como se mueve, juega con el OS.
