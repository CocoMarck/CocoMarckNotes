# Tipos de distribuciones GNU linux. Sistemas operativos GNU-Linux


Existen muchisimas distribuciones GNU-Linux.
Veremos primero las mas canijas (dificiles de usar) a las mas sencillas.

## Distros avanzadas o super avanzadas
Existen distros para usuarios avanzados, que para usarlas bien, necesitas tener conocimientos intermedios-avanzados para su uso. Hasta a veces tienes que programar y toda la cosa.

**Ejemplos:**
- **Arch Linux**. Ta dificil, mucho texto.
- **Gentoo**. Se tiene que compilar todo, hasta la hora.




## Distros intermedias
Tienes que tener conocimientos basicos del funcionamiento de GNU-Linux, tendras que usar terminal en muchas ocasiones.

**Ejemplos:**
- Debian. 
- CachyOS.
- OpenSuse.
- Fedora.
