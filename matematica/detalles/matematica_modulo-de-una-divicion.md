# Modulo de una división

## Ejemplos de como obtener el modulo de una divición

### Sacar modulo:

$Dividendo -(Cociente \cdot Divisor) = ModuloDeDivición$

**El modulo de quince entre diez es cinco:**
```
    1.5
10/15
   10
    5 <-- Modulo, resto de divición
```

> El cociente es $1$, ya que el valor de la divición es $1.5$, tomamos unicamente como cociente el valor entero, no el flotante.

> $15 -(10 \cdot 1) = 5$ `<-- Modulo, resto de divición`

    

**El modulo de Diez entre quince es diez:**
```
    0.6
15/10 0
    9 0
    1 0 <-- Modulo, resto de divición
```

> El cociente es $0$, ya que el valor de la divición es $0.6$, tomamos unicamente como cociente el valor entero, no el flotante. 

> $10 -(15 \cdot 0) = 10$ `<-- Modulo, resto de divición`
   


**El modulo de Treinta entre veinte es diez:**
```
    1.5
20/30
   10 <-- Modulo, resto de divición
```
   
> El cociente es $1$, ya que el valor de la divición es $1.5$, tomamos unicamente como cociente el valor entero, no el flotante.

$30 -(20 \cdot 1) = 10$ `<-- Modulo, resto de divición`
