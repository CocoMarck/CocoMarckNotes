Por que `x` sobre `z`, esto sobre `y` sobre `z`, es igual a `x` sobre `y`.

${x : z} \over {y : z}$

= ${ (x : 1) \times (1 : z) } \over { (y : 1) \times (1 : z) }$

= ${ x \cdot (1 : z) } \over { y \cdot (1 : z) }$

= ${ x \cdot [ (1:z) : (1:z) ] } \over y$

= $( x \cdot 1 ) \over y$

= $x \over y$
