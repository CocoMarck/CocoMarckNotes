# Operar fracciones de forma cruzada
Solo aplica para sumas de fracciones y para restas de fracciones.


## Para sumas de fracciones:
${x \over y} + {z \over w} $

=   ${ xw + zy } \over {yw}$

=   ${ (x \cdot w) + (z \cdot y) } \over {y \cdot w}$


## Para restas de fracciones:
${x \over y} - {z \over w}$

=   ${ xw -zy } \over {y \cdot w}$

=   ${ (x \cdot w) -(z \cdot y) } \over {y \cdot w}$
