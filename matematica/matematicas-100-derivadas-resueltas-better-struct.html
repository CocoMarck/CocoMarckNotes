Ejercicio 80

$$ y = x \cdot \sqrt{4 -x^2} + 4arcsen({x \over 2}) $$

"x" por la raiz de cuatro menos "x" al cuadrado, esto mas cuatro arcoseno de "x" sobre dos

$$ y' = \sqrt{4 -x^2} + { {x \cdot -2x} \over {2 \sqrt{4 -x^2} } } + 4 \cdot { { 1 \over 2 } \over { \sqrt{ 1 -{x \over 2}^2 } } } $$ $$ = \sqrt{4 -x^2} + { {x \cdot {-2x \over 2} } \over {\sqrt{4 -x^2} } } + 2 \cdot { { 2 \cdot {1 \over 2} } \over { \sqrt{ 1 -{x \over 2}^2 } } } $$

Primeramente la regla de derivación del producto. Para: `$x \cdot \sqrt{4 -x^2}$`, es `$\sqrt{4 -x^2}$`.

  • Regla del producto: $x \cdot y = x \cdot y' + y \cdot x'$.
